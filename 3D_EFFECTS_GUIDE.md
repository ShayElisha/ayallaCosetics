# 🎨 מדריך לאפקטים התלת-מימדיים באתר

## 🌟 מה הוספנו?

האתר שלך עכשיו כולל אפקטים תלת-מימדיים מתקדמים שיוצרים חוויה ויזואלית מדהימה!

---

## 🎯 אפקטים לפי סקציה:

### 1️⃣ עמוד הבית (Hero)

**אפקטים:**
- ✨ אלמנטים מרחפים ברקע שמסתובבים בתלת מימד
- 🎪 Parallax scrolling - האלמנטים זזים במהירויות שונות כשגוללים
- 💫 כפתורים עם אפקט הרמה והטיה כש-hover
- 🌈 כיתוב ראשי עם עומק תלת-מימדי

**איך זה עובד:**
- כשגוללים למטה, האלמנטים זזים בקצבים שונים (Parallax)
- העכבר מעל הכפתורים גורם להם "לצוף" ולהתכופף קלות

### 2️⃣ שירותים (Services)

**אפקטים מתקדמים:**
- 🎴 כרטיסים שמגיבים לתנועת העכבר
- 🔄 האייקונים מסתובבים 360 מעלות כש-hover
- 📐 כל אלמנט בכרטיס נמצא בעומק אחר (depth layers)
- ✨ אפקט ברק (shine) שעובר על הכרטיס

**איך זה עובד:**
- העכבר מעל הכרטיס גורם לו להתכופף בכיוון העכבר (3D tilt)
- כל חלק בכרטיס נמצא בעומק שונה:
  - האייקון: העומק הכי גדול (100px)
  - הכפתור: עומק בינוני (60px)
  - הטקסט: עומק קטן (30-50px)

### 3️⃣ גלריה (Gallery)

**אפקטים:**
- 🖼️ תמונות שמגיבות לתנועת העכבר עם הטיה תלת-מימדית
- 💨 אנימציית כניסה עם סיבוב
- ✨ אפקט ברק שעובר על התמונה
- 🎭 שכבות עומק שונות (תמונה, overlay, תג)

**איך זה עובד:**
- כל תמונה מגיבה לתנועת העכבר ונוטה בכיוון ההפוך
- כשעוברים עם העכבר, יש אפקט של "ברק" שעובר על התמונה

## 🛠️ טכנולוגיות שבשימוש:

### Framer Motion
```javascript
// דוגמה לאפקט תלת-מימד
<motion.div
  whileHover={{ 
    rotateX: 5,
    rotateY: 5,
    z: 50 
  }}
  style={{ 
    transformStyle: 'preserve-3d',
    transform: 'translateZ(50px)' 
  }}
>
```

### CSS 3D Transforms
```css
.card-3d {
  transform-style: preserve-3d;
  perspective: 1000px;
}

.depth-1 {
  transform: translateZ(25px);
}
```

## 🎮 אפקטים שמגיבים לפעולות:

### 1. Mouse Move (תנועת עכבר)
- **כרטיסי שירותים**: נוטים בכיוון העכבר
- **גלריה**: תמונות מתכופפות עם העכבר

### 2. Hover (מעבר עכבר)
- **כפתורים**: עולים למעלה ונוטים קלות
- **כרטיסים**: מתרוממים מהדף
- **אייקונים**: מסתובבים 360 מעלות

### 3. Scroll (גלילה)
- **Hero**: אלמנטים זזים במהירויות שונות
- **Background**: צורות מתרחבות וזה

### 4. Click (לחיצה)
- **כפתורים**: אנימציית "לחיצה" (scale down)
- **גלריה**: פתיחת Lightbox עם אנימציה

## 📐 שכבות עומק (Depth Layers):

המערכת משתמשת ב-`translateZ` ליצירת עומק:

```
📊 רמות עומק:
├─ 100px - אלמנטים חשובים מאוד (אייקונים)
├─ 75px  - אלמנטים חשובים (כותרות)
├─ 50px  - אלמנטים רגילים (טקסטים)
├─ 25px  - רקע קל (borders, effects)
└─ 0px   - רקע בסיסי
```

## 🎯 דוגמאות לשימוש:

### הוספת אפקט תלת-מימד לאלמנט חדש:

```jsx
<motion.div
  whileHover={{ 
    y: -10,           // עולה למעלה
    rotateX: 5,       // נוטה קדימה
    rotateY: 5,       // נוטה הצידה
    scale: 1.05       // מתרחב
  }}
  style={{
    transformStyle: 'preserve-3d',
    perspective: '1000px'
  }}
>
  התוכן שלך כאן
</motion.div>
```

### הוספת עומק לאלמנטים פנימיים:

```jsx
<div style={{ transform: 'translateZ(50px)' }}>
  <h3>כותרת עם עומק</h3>
</div>

<div style={{ transform: 'translateZ(100px)' }}>
  <button>כפתור שצף למעלה</button>
</div>
```
